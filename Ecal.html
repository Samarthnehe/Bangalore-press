The Hindu / Chandramana / Sowramana / Muslim calendar and festivals and events will now be available automatically on your desktop calendar.

  • Available in Kannada

What's Continues??

Install Requirements

Windows:

Just run the installer and follow the instructions.

Mac OS X:

Double click on the .dmg file and copy the file to your Applications folder.

Linux (Ubuntu/Debian):

- Download the DEB version of Rainlendar2

- Double click on the package

- Click "Install package"

You can start Rainlendar from the Applications->Office menu.
